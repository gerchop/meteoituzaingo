# Meteo Ituzaingó

## Descripción del proyecto

Meteo Ituzaingó es un sitio meteorológico desarrollado para mostrar en tiempo real
los datos provenientes de una estación Weather Underground (Weather.com PWS).

El proyecto se publica mediante GitHub Pages y posteriormente se integra dentro
de un Blog de Blogger.

El objetivo NO es únicamente mostrar datos meteorológicos.

El objetivo es construir el mejor sitio meteorológico local de Argentina,
con una experiencia de usuario comparable con:

- Windy
- AccuWeather
- Weather Underground
- Meteored

utilizando exclusivamente los datos propios de la estación.
